# Modelo 3D LABPRO
Este repositório contém o modelo 3D do LABPRO. Modela a sala atual com a bancada, pia e espaços de trabalho.

# Como abrir
1. Obtenha e instale o programa [SweetHome3D](https://www.sweethome3d.com)
2. Execute o SweetHome3D
3. Abra o arquivo
